# MedTrack+ Web

Aplicación web de alarmas de medicamento desarrollada en Angular.

## Ejecución

Usar Node 22 LTS.

```bash
source ~/.nvm/nvm.sh
nvm use 22
cd medtrack-web
npm install
npm start
```

La aplicación web se levanta normalmente en:

```text
http://localhost:4200/
```

## Build

Para generar el build de producción

```bash
cd medtrack-web
npm run build
```
